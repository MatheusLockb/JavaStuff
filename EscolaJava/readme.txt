# Escola-Java

## Descrição do Projeto
**Escola-Java** é um aplicativo desktop desenvolvido em **Java Swing** para gerenciar de forma abrangente instituições de ensino, incluindo escolas, professores, alunos e cursos. Este projeto visa proporcionar uma solução prática e eficiente para o gerenciamento educacional, facilitando a organização e a administração das informações.

## Funcionalidades Principais
- **Gerenciamento de Escolas**: Criar e editar informações detalhadas de cada escola.
- **Cadastro de Diretores**: Adicionar e associar diretores a cada escola.
- **Cadastro de Professores e Alunos**: Adicionar professores e alunos a uma escola específica, com possibilidade de edição e exclusão.
- **Criação e Gestão de Cursos**: Criar cursos com detalhes, associar professores e alunos, e gerenciar o conteúdo de cada curso.

## Tecnologias Utilizadas
- **Linguagem de Programação**: Java (Java SE)
- **Interface Gráfica**: Java Swing
- **Persistência de Dados**: (opcional) JDBC para conexão com bancos de dados
- **Ferramentas de Build**: Maven/Gradle (dependendo das preferências do desenvolvedor)

## Requisitos do Sistema
- **Java JDK 8** ou superior
- **IDE de Desenvolvimento** (IntelliJ IDEA, Eclipse, NetBeans ou outro)
- Dependências externas mencionadas no `pom.xml` ou `build.gradle` (caso use Maven ou Gradle)
